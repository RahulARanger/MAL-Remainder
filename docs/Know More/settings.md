# Features

In Settings page one can 

* [Force a Remainder](#Force-A-Remainder)
* ReSchedule events from the calendar
* Reset OAuth
* Refresh Tokens
* Import Settings


And other it redirects to your AnimeList in [MyAnimeList](https://myanimelist.net)/animelist/.
